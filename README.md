# Student Attendance Certification System

## Project Overview

This project analyzes student attendance data and identifies students eligible for certification.

A student is considered **Certified** if they attend **32 or more sessions out of 40 sessions**. Otherwise, the student is marked as **Not Certified**.

## Objectives

* Collect attendance data of students
* Perform data cleaning and preprocessing
* Calculate attendance count for each student
* Apply certification eligibility logic
* Visualize attendance and certification results
* Generate a final list of certified students

## Dataset

* Total Students: 980
* Total Sessions: 40
* Attendance Values:

  * 1 = Present
  * 0 = Absent

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Certification Logic

If Attendance Count >= 32:
Certified

Else:
Not Certified

## Project Workflow

1. Load attendance dataset
2. Perform data cleaning
3. Calculate attendance count
4. Apply certification logic
5. Generate certified and non-certified student lists
6. Create visualizations
7. Save final results
